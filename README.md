### 上手指南





###### 开发前的配置要求

1. Nodejs >= v14
2. 微信开发者id

###### **安装步骤**

1.Clone the repo

```sh
git clone https://github.com/weihyZzz/weihyMusic.git
```

### 部署

1.安装npm依赖包
```
npm install
```
2.微信开发者工具 => 工具 => 构建npm

3.编译执行

### 使用到的框架

- [原生微信小程序](https://developers.weixin.qq.com/miniprogram/dev/framework/)
- [vant](https://github.com/youzan/vant)
